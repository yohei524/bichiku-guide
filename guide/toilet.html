1. なぜトイレが最優先なのか

地震で水道止まる → 水洗トイレ流せない → 3時間後から使えなくなる。そのまま使うと詰まる・溢れる。マンション高層階なら下の階まで被害いく。これで隣人関係終わる。

  • 成人1人が1日5〜7回トイレ行く
  • 水が止まって24時間で我慢限界
  • 女性・子供は特にストレス激しい
  • 避難所のトイレは大行列＆衛生最悪
⚠ 在宅避難の成否はトイレで決まる。食料より水より、トイレがないと家に居られなくなる。結果、避難所行き → 感染症・ストレス・盗難リスク。

3. 簡易トイレの種類

凝固剤タイプ（王道）

便座に袋をかぶせて、用を足したら凝固剤をふりかけてゲル化→縛って捨てる。処理がラク。まずこれ。

吸水シートタイプ

シートが水分を吸う。凝固剤より割高だが臭いが少ない傾向。

ポータブル便座型

便座自体が無い環境用（車中泊・アウトドア兼用）。普段使わないなら折りたたみ式で十分。

簡易トイレの定番

簡易トイレ 50回 簡易トイレ 100回 マイレット（人気） 凝固剤単品 折りたたみポータブル便座

4. マンション・高層階の絶対条件

ここは命に関わる話。絶対に手を抜かないで。

何が起きるか

  • エレベーター停止 → 階段で下まで降りられない（高齢者・子供）
  • 排水管破損 → 流したら下階が汚水まみれ
  • 上階で誰かが流す → 自分の部屋に逆流するケースも
  • 避難所まで徒歩が遠い → そもそも間に合わない
⚠ マンション管理組合から「流すな」指示が出るケースが多い。阪神淡路・東日本・熊本・能登、全部で起きてる。簡易トイレがなければ家で生活できなくなる。

マンション住民の必須装備

  • 簡易トイレ（人数×7日×5回が最低ライン）
  • 消臭袋（BOS等） ── 臭いが防げないと精神的にキツい
  • 45L以上のゴミ袋（溜まった使用済み袋を一時保管）
  • ダンボール便座（便座が使えない時用）

マンション高層階の必須セット

簡易トイレ100回 BOS消臭袋 45Lゴミ袋 ダンボール便座

5. 戸建て・庭がある人向け

庭があるなら話が全然違う。穴掘って埋めるのが最強。昔はみんなこれやった。

やり方

  1. 庭の隅に深さ40cmくらいの穴を掘る
  2. 用を足したら土をかぶせる
  3. 穴が埋まったら隣を掘る
  4. 完全に分解されるまで半年〜1年

スコップ1本で全部解決。断水何日続いても余裕。匂いも気にならん（土が吸う）。戸建ての強み活かしたいならコレ。

⚠ 水道管・ガス管のある場所は避けて。また上水道の井戸がある家は離れた場所で。

戸建て向け

園芸用スコップ 折りたたみスコップ 予備の簡易トイレ

6. 処理の仕方（臭い対策）

使用済み袋を溜めとく時、臭い対策しないと家が地獄になる。

  • BOS消臭袋（医療用レベル）は別格。5年経っても臭わない
  • 使用済み袋 → BOS → 45Lゴミ袋 → ベランダで二重三重にする
  • 夏場は特に要注意。直射日光避けて冷暗所に
  • ゴミ収集再開まで数週間かかることを想定

臭い対策

BOS LLサイズ BOS SSサイズ 蓋付きゴミ箱
